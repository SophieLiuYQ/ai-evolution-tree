## Specialized variants in the Qwen 2.5 family

| Variant | Sizes | Specialization |
|---|---|---|
| Qwen 2.5 (base) | 0.5B-72B | general purpose |
| Qwen 2.5-Instruct | 0.5B-72B | chat / instruction following |
| Qwen 2.5-Coder | 0.5B-32B | code generation; Coder-32B competitive with GPT-4o-mini on coding |
| Qwen 2.5-Math | 1.5B / 7B / 72B | math reasoning |
| Qwen 2.5-VL | 3B / 7B / 72B | multimodal vision |

The Coder and Math variants exemplify a productization pattern Western labs have largely avoided — domain-specific specialized model variants instead of one general flagship.

## The "AI tigers" funding context

By Qwen 2.5's release, Alibaba had positioned itself differently from China's "AI tiger" startups (Zhipu, Moonshot, Baichuan, MiniMax). Where the tigers raised hundreds of millions for narrow flagship products, Alibaba treated AI as infrastructure investment — open Qwen as ecosystem fuel, monetized through cloud (Aliyun GPU rental, model API, fine-tuning services) rather than direct model sales. This positioning turned out to be more sustainable as the AI tigers struggled with monetization in 2025.
