## What it is

A combined system from Google DeepMind for solving IMO-level mathematical problems. Two components:

1. **AlphaProof** — handles algebra and number theory problems via formal proof in Lean 4
2. **AlphaGeometry 2** — handles geometry problems via a different (geometry-specialized) formal system

For AlphaProof: a Gemini-derived language model translates natural-language problems into Lean 4 statements, then iteratively proposes proof attempts. Lean 4's type checker provides ground-truth verification: a proof either compiles or doesn't. RL training optimizes the LM against this verifiable reward.

The reasoning traces are extensive — solving a single IMO problem can take 1-3 days of compute (parallel search + RL fine-tuning per problem). This is "extreme inference-time compute" — much more than even o1-pro consumes per query.

## IMO 2024 results

| Problem | Type | AlphaProof / AlphaGeometry result | Time |
|---|---|---|---|
| 1 | Algebra | ✓ correct proof | minutes |
| 2 | Algebra | ✓ correct proof | minutes |
| 3 | Combinatorics | ✗ unsolved | exceeded budget |
| 4 | Geometry (AlphaGeometry 2) | ✓ correct proof | seconds |
| 5 | Combinatorics | ✗ unsolved | exceeded budget |
| 6 | Algebra | ✓ correct proof (the hardest problem of the year) | days |

Total: 28/42 points — silver medal threshold (top ~150 of ~600 participating students). Gold medal threshold (top ~50) was 29 points, which AlphaProof missed by 1 point.

Notably: problem 6 was solved by AlphaProof but only ~5 of 600 human students did so. Problem 3 (combinatorics) defeated AlphaProof; combinatorics is typically the hardest area for formal-proof systems because the combinatorial search space is large.

## Lineage and influence

AlphaProof's technique — neural network proposes, formal system verifies, RL on the verified-correct examples — is essentially the same recipe DeepSeek R1 used 6 months later for general math/code reasoning (with code-execution / answer-checking as the verifier instead of Lean). The methodology transferred from "specialized formal-math system" to "general reasoning model" within roughly a year — one of the faster technique-transfer rates in the modern LLM era.
