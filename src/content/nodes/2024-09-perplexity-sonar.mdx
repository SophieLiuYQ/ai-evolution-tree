## What it is

A family of LLM API endpoints from Perplexity AI optimized for grounded question-answering with citations. Three sizes:

| Variant | Base | Specialization |
|---|---|---|
| Sonar Small | Llama 3.1 8B | fast Q&A with web grounding |
| Sonar Large | Llama 3.1 70B | higher-quality answers |
| Sonar Huge | undisclosed (~Llama 3.1 405B class) | flagship, longest reasoning |

The post-training and serving infrastructure are Perplexity's contributions, not the base models themselves. The combined system (LLM + retrieval + citation post-processing) is what's productized.

## Perplexity vs concurrent "AI search" peers

| Product | Date | Approach |
|---|---|---|
| Bing Chat / Copilot | 2023-02 | GPT-4 + Bing search |
| Google AI Overviews | 2024-05 | Gemini + Google Search; default-on |
| **Perplexity (consumer + Sonar API)** | **2022-12 / 2024-09** | **dedicated AI-search product + API** |
| You.com | 2022 | early entrant, struggled to differentiate |
| Phind | 2022 | dev-focused AI search |
| ChatGPT Search | 2024-10 | OpenAI's response — search inside ChatGPT |
| Claude Web Search | 2025-Q2 | Anthropic's response |

Perplexity's bet: search-as-primary-product can sustain a standalone company despite Google's distribution and OpenAI's brand. By 2026 the bet is partly working (revenue scaling, valuation holding) and partly under pressure (consumer AI-search has become table-stakes for every chat product).
