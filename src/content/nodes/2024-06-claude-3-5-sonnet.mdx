## What it is

A mid-tier model in the Claude 3.5 generation. At launch (June 20, 2024), the only model in the 3.5 line; Anthropic added Claude 3.5 Haiku (October 2024) but never released a 3.5 Opus, instead jumping to Claude 3.7 Sonnet and eventually Claude 4 Opus.

Quality position: matched or beat both GPT-4o and Claude 3 Opus on most reasoning, math, and coding benchmarks while priced identically to Claude 3 Sonnet ($3/$15 per 1M tokens — i.e., ⅕ the cost of Opus).

## Artifacts — a UX shipping co-product

Released alongside the model: the "artifacts" feature in claude.ai split the conversation into a chat panel (left) and a rendered output panel (right) that automatically displays generated code, UI mockups, SVG diagrams, or documents. This UX pattern was almost immediately copied by ChatGPT (Canvas, October 2024), Gemini (Canvas, late 2024), and several agentic IDEs.

The product lesson: model quality alone wasn't enough to win developer mindshare; the way model output was surfaced to the user mattered just as much. Anthropic's product velocity in 2024-25 — artifacts, projects, computer use, MCP — is a key part of the Claude commercial story not captured by benchmark deltas.
