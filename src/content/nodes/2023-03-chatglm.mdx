## How ChatGLM compared to peers (March 2023)

| Model | Date | Org | Params | Chinese | License |
|---|---|---|---|---|---|
| GPT-4 | 2023-03-14 | OpenAI | undisclosed | strong | API only |
| Claude 1 | 2023-03-14 | Anthropic | undisclosed | weaker | API only |
| **ChatGLM-6B** | **2023-03-14** | **Tsinghua** | **6.2B** | **native** | **research-only** |
| Llama 1 | 2023-02-24 | Meta | 7B-65B | weak | research-only |
| Bard | 2023-03-21 | Google | undisclosed | weak | free chat |

ChatGLM was the only open release in this group with native Chinese capability. The strategic positioning — bilingual + small + open — defined the Chinese open-LLM playbook.

## Lineage to GLM-4

The ChatGLM line evolved into:
- **ChatGLM2-6B** (Jun 2023) — 32k context, 12% better quality
- **ChatGLM3-6B** (Oct 2023) — function calling, code interpreter
- **GLM-4** (Jan 2024) — closed flagship, multimodal, claims GPT-4-class on Chinese
- **GLM-4-9B** (Jun 2024) — open mid-tier
- **GLM-4-Plus** (2025) — ongoing
