## Llama 2 Chat — open-source RLHF in production

Meta published the full RLHF pipeline used for Llama 2 Chat: SFT on 27.5k high-quality demonstrations, then iterative reward modeling and rejection-sampling fine-tuning (RFT) for several rounds, then PPO. This was the first detailed open-source description of a frontier-quality RLHF training run — and it set the methodological template that Mistral, Qwen, DeepSeek, and dozens of others followed.

Reward model architecture: two reward models — one for helpfulness, one for safety — trained on 1.4M human-labeled preference comparisons. The two-headed approach helped Llama 2 Chat avoid the "over-refusal" problem that plagued early RLHF deployments.
