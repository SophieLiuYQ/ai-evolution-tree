## How it differs from InstructGPT/RLHF

| Dimension | InstructGPT (2022-01) | Constitutional AI (2022-12) |
|---|---|---|
| Preference source | human raters | model critique against text rules |
| Cost per data point | $0.20–$2 | ~free (compute only) |
| Auditability | preference data | preference data + constitution |
| Bottleneck | annotator hiring | reward-model + policy quality |
| Scalability | linear in $$$ | linear in compute |
| Adopter | OpenAI | Anthropic |

Both are now used together at all major labs — RLHF for some objectives, RLAIF / CAI for others.

## What's in Anthropic's actual constitution

Public-facing principles (excerpts):
- "Please choose the response that is most supportive and encouraging of life, liberty, and personal security."
- "Please choose the response that has the least objectionable, offensive, unlawful, deceptive, inaccurate, or harmful content."
- "Please choose the response that's least intended to build a relationship with the user."

The full constitution has been iteratively refined and is partially public. The opacity is itself a critique vector: "auditable" only goes so far if the operative version evolves with each model release.
